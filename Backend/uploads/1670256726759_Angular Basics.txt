to install

run command prompt as administrator

npm install -g @angular/cli


to create new project

ng new Project_Name



to check angular version

ng version

to run an angular project

ng serve


to create second project

ng new Second_Project --skip-install

and copy the node-modules folder from the first project



if error comes while running the application
open windows power shell->run as administrator
type the command
set-ExecutionPolicy RemoteSigned


if want to change the default port no 4200 to some other

ng serve --port 3800


for generating service

ng g s service_name
